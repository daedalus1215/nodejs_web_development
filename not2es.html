Dynamic Example

In order to do a dynamic EJS templating example we must use <%= %> to embed our javascript. Below is a great example of how we use it.

In the index.js file we would have a route that looks like: app.get('/fallin/:thing', function(req, res) { var thing = req.params.thing; res.render("love.ejs", {thingVar: thing}); }); Then inside of love.ejs we would have something like:

You fell in love with: <%= thingVar %>
